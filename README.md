# ReposteriaCalc

Calculadora de costos de repostería - Calcula el costo de tus recetas de galletas, tortas y postres.

## Funciones

- Calcula costo de materia prima por receta
- Conversión automática de unidades (g, kg, ml, L)
- Historial de recetas guardadas
- Diseño responsive y modo oscuro
- App instalable (PWA)

## Uso

1. Abre en un navegador
2. Ingresa el nombre de la receta
3. Agrega los ingredientes con precio y cantidad
4. Define el margen de ganancia
5. Guarda las recetas para reutilizarlas
